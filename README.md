# Coinly - Track Your Wealth

Coinly is a modern money tracker application built with Next.js, TypeScript, and MongoDB (Mongoose). It allows users to track expenses, income, and manage their payment methods with a sleek, responsive interface.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT-based with Cookies

## Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up environment variables**:
   Create a `.env` file based on `.env.example`.
4. **Run the development server**:
   ```bash
   npm run dev
   ```

## Key Features
- **Dashboard**: Overview of current balances and financial summary.
- **Ledger**: Detailed transaction history with spreadsheet-style editing.
- **Profile**: Manage user account and payment methods.
- **Modern UI**: Dark-mode compatible, premium aesthetics with Tailwind CSS.
